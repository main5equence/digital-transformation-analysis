# Digital Transformation Leaders Analysis

This project explores activity patterns in a large-scale digital transaction system using exploratory data analysis (EDA).

## Tools

- Python
- Pandas
- NumPy
- Seaborn
- Matplotlib
- Jupyter Notebook

## Project Goals

The objective of this project is to analyze transaction behavior, event distribution, and client activity patterns within a digital platform environment.

## Key Insights

- Transaction values show a highly skewed distribution.
- A small number of clients generate a large portion of system events.
- Transaction activity remains relatively stable across most months.
- Certain event types dominate system interactions.
